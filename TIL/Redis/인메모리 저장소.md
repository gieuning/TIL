# 인메모리 저장소

데이터를 하드 디스크 대신 컴퓨터의 메모리(RAM)에 직접 저장하는 방식.
데이터가 이미 메모리에 있기 때문에 읽고 쓰는 속도가 빠르다.

RAM은 CPU 가까이 있으며, CPU 연산을 보조하는 역할을 한다.

## 활용 사례
- **캐싱**: 자주 조회되는 데이터를 메모리에 미리 저장해두면 빠르게 반응할 수 있다.
- **세션 관리**: 로그인 정보, 장바구니 등 세션 데이터를 메모리에 저장해 빠르게 처리한다.
- **실시간 데이터 처리**: 주식 시세, 게임 랭킹, 실시간 알람처럼 순식간에 변하는 데이터를 처리하는 데 유용하다.

## 특성
- 속도는 빠르지만, 데이터의 영구적 보존에는 비용이 높다.
- **휘발성 메모리**: 시스템이 꺼지면 데이터가 유실된다.
- 인메모리는 양날의 검 → 잘 쓰면 성능이 향상되지만, 잘못 쓰면 장애 요소가 된다.
